# game

Основной репозитарий игры Null Terminal.

[Лор игры](LORE.md)
[Заметки разработки](BLOG.md)

## Как собрать?

```bash
npm install
npm run dev:game
npm run dev:editor
```

Далее у вас запуститься разработческий сервер, где можно загрузить нужный модуль.

## Модули

1. Редактор спрайтов - [sprite-editor.html](sprite-editor.html).
   Отвечает за редактирование, нарезку спрайтов и анимации.

2. Игра - [game.html](game.html).
   Песочница самой игры.
